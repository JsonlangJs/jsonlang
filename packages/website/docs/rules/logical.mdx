# Logical

All logical operations

``` js 
import { JsonLang } from '@jsonlang/core';
import { LogicRules } from '@jsonlang/logic';

const jsonLang = new JsonLang();

jsonLang.import(LogicRules);
```

## And or && 
  * **Input[]:** Array<any\> (Size: Unlimited). Array or any data or rules.
  * **Output:** Boolean (true or false).
  * **Description:** Do the `Anding` operation, if any value in `Input[]` has a value of (null, 0, false), it will return `false`, else it will return `true`.

## Or or ||
  * **Input[]:** Array<any\> (Size: Unlimited). Array or any data or rules.
  * **Output:** Boolean (true or false).
  * **Description:** Do the `Oring` operation, if all values in `Input[]` has a value of (null, 0, false), it will return `false`, else it will return `true` .
<JsonLangRunner example="9" height="13vh" key="example9"></JsonLangRunner>

## Equal or ==
  * **Input[]:** Array<any\> (Size: 2). Two fields with any data type or rules to compare.
  * **Output:** Boolean (true or false).
  * **Description:** It takes an array of 2 inputs to compare if element one `Equal` element two or not. If the provided fields have different data types, it will cast them to same data type to compare.

## If-Else
  * **Input[]:** Array<Mixed\> (Size: 3), 
    * Condition: it can be Value or Rule and if it succeeded `(true, value)` it will call `Then`, but if it failed `(false, null, 0)` it will call `Else`
    * Then: it can be Value or Rule.
    * Else: it can be Value or Rule.
  * **Output:** any depends on the value of `then` or `else`.
  * **Description:** to do the if-else logic, all the 3 inputs are required, if u didn't pass one of them it will return `false`.
