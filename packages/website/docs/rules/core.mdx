# Core

Access data in different scopes

``` js 
import { JsonLang } from '@jsonlang/core';

const jsonLang = new JsonLang();
```


## Data  or getData
  * **Input[]:** Array<string\> (Size: 2) 
    * Variable Name: 1st argument, if its value is `''` or `'*'`, it will return the whole data, but you can also specify the variable name you want to get
    * Type: Enum of "External" or "Internal". External to get the schemaless `data` object which you pass it to the [**Execute**](../intro#execute) method. Internal to get the value of any [**Output**](../intro#structure) from any rules, or the value passed from the parent rule like [**Filter**](./array#filter).
  * **Output:** any.
  * **Description:** Access External or Internal to Read/Write.
